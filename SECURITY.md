# Security Policy

## Supported Versions

We apply security fixes to the current release only. Older versions are not
backported.

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email **security@f7-platform.io** with:

- A description of the vulnerability
- Steps to reproduce (proof of concept if available)
- The versions/platforms affected
- Any mitigations or workarounds you have found

You will receive an acknowledgement within 3 business days. We aim to release
a fix within 14 days of confirming the issue; if that timeline cannot be met
we will communicate the adjusted schedule.

We follow a 90-day coordinated disclosure policy. We ask that you give us
that window before disclosing publicly.
