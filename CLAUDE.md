# public-agent-binaries

Public repository containing pre-built F7 Agent installer binaries for all supported platforms.

## Structure

- `v{version}/` — Versioned directories containing platform-specific binaries
- `README.md` — Download and installation guide for IT admins

## Key Facts

- Binaries are built by the `fseven-agent` CI release pipeline
- Each binary includes: `.pkg` (macOS), `.msi` (Windows), `.tar.gz` (Linux)
- SHA-256 checksums are published alongside each binary
- Ed25519 signatures are verified by the agent at update time
- This repo is public-facing — never commit secrets, internal docs, or pre-release builds

## Pre-Push CI Gate

This repo has no build system, but before pushing verify:
- Binary files are in the correct `v{version}/` directory structure
- No secrets, internal docs, or pre-release builds are included
- SHA256SUMS will be auto-generated by CI after push

## Build Commands

This repo has no build system — it is a binary distribution repository.
